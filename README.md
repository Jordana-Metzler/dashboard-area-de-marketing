## Sobre o projeto

Este projeto foi desenvolvido durante o curso **Microsoft Power BI para Business Intelligence e Ciência de Dados** e consiste em um dashboard interativo para análise de dados da área de Marketing.

O relatório foi estruturado em quatro páginas — **Cliente**, **Comportamento do Cliente**, **Campanhas** e **Pontos de Venda** — com o objetivo de responder perguntas de negócio relacionadas ao perfil dos clientes, comportamento de compra e efetividade das campanhas de marketing.

Durante o desenvolvimento, foram aplicados conceitos de modelagem de dados, criação de medidas em DAX e construção de dashboards interativos. Além disso, o projeto proporcionou a exploração de diferentes tipos de visualizações, como **matrizes**, **gráficos de dispersão** e **árvores hierárquicas**, ampliando as possibilidades de análise para além dos gráficos tradicionais.
